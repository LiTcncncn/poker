## 🚀 系统功能说明

### 1. 玩法系统
- **52种独特玩法**：从♠️A到♦️K，每张超级牌对应一个玩法
- **多维度组合**：
  - 张数：3/4/5/6张
  - Joker概率：0%/4%/8%
  - 特殊规则：n张成顺/成花、单数/双数顺子、花色限制、赔率加成
  - 任务类型：一对、两对、三条、顺子、同花、葫芦等

### 2. 超级牌解锁机制
- **顺序解锁**：必须从♠️A到♦️K依次解锁
- **三重条件**：
  1. 规则满足：在指定玩法下游戏
  2. 任务完成：完成指定次数的牌型（如"顺子3次"）
  3. 费用支付：一次性支付$（100→2650递增）
- **进度追踪**：实时显示任务进度、费用状态
- **自动推进**：解锁后自动初始化下一张超级牌

### 3. 特殊规则支持
- **n张成顺/成花**：3张或4张玩法也能识别顺子/同花
- **单数/双数顺子**：只允许单数或双数的顺子
- **花色限制**：限制同花必须是红色或黑色
- **赔率加成**：特定牌型获得额外赔率

### 4. 成就系统限制
- **5张牌限制**：顺子、同花、同花顺、皇家同花顺只有5张牌才计入成就
- **3/4张玩法**：不影响这些成就的进度
- **其他牌型**：一对、两对、三条等不受张数限制

## 📝 使用指南

### 设置玩法
```typescript
import { useGameStore } from './store/gameStore';

const { setGameMode } = useGameStore();

// 设置当前玩法
setGameMode('super_3_card'); // 超级三张玩法
```

### 查询当前玩法
```typescript
const { getCurrentGameMode } = useGameStore();

const gameMode = getCurrentGameMode();
if (gameMode) {
  console.log('当前玩法:', gameMode.name);
  console.log('翻牌数量:', gameMode.cardCount);
  console.log('Joker概率:', gameMode.jokerProbability);
}
```

### 支付解锁费用
```typescript
const { paySuperCardUnlockCost } = useGameStore();

// 支付费用
const success = paySuperCardUnlockCost('spades_A', 100);
if (success) {
  console.log('支付成功');
}
```

### 解锁超级牌
```typescript
const { unlockSuperCardWithTask } = useGameStore();

// 解锁超级牌（需要满足所有条件）
const success = unlockSuperCardWithTask('spades_A');
if (success) {
  console.log('解锁成功！');
}
```

## 🔧 技术细节

### 数据持久化
- **玩法配置**：存储在 `src/config/gameModes.json`
- **解锁进度**：localStorage (`supercard_unlock_progress`)
- **已解锁列表**：localStorage (`unlocked_super_cards`)
- **任务进度**：gameStore (`gameModeTaskProgress`)

### 解锁顺序控制
```typescript
// 在 unlockSuperCard() 中自动检查
if (unlock.unlockOrder > 1) {
  const prevUnlock = getSuperCardUnlockByOrder(unlock.unlockOrder - 1);
  if (prevUnlock && !unlockedCards.includes(prevUnlock.superCardId)) {
    console.error('必须按顺序解锁超级牌');
    return false;
  }
}
```

### 任务进度同步
```typescript
// flipCards 中自动更新
if (currentGameModeId && currentGameMode) {
  const { updateGameModeTaskProgress } = get();
  updateGameModeTaskProgress(result.type);
}

// 同步到超级牌解锁进度
if (gameMode.isSuperCardUnlock && gameMode.superCardId) {
  updateSuperCardTaskProgress(gameMode.superCardId, 1);
}
```

### 成就5张牌限制
