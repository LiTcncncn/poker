# Roguelike 超级牌（策划稿）

> **用途**：玩法与数值口径；与 `roguelike/` 实现同步目标。**局内 💎** 为商店购买代币；**本局失败** 后该局 💎 与局内进度作废（不保留到局外）。

---

## 0. 局 / 关 / 手（统一用语）

| 概念 | 含义 |
|------|------|
| **局（Run）** | 玩家从开局到 **胜利通关** 或 **冲关失败** 的一次完整流程；含主线与 **无限模式** 等。 |
| **关（Stage）** | 一局内由多个关卡组成；每关有目标金币、手数上限、词缀等。 |
| **手（Hand）** | 一关内玩家完成 **翻牌 →（可补牌）→ 结算** 的一次操作；一关可打多手。 |

---

## 1. 名词

- **超级牌**：界面用语；数据里即 **属性牌**，购入后进牌池，与 52 张基础牌一起洗。
- **品质**：商店随机超级牌仅 **绿 / 蓝 / 紫**（`super` 等为遗留枚举，商店不产出）。
- **「+$N」**：在牌面点数底分上再 **+N**（`high_score`）。
- **「+N 倍」**：参与计分时 **+N 倍率**（`multiplier` + 牌张 `card.multiplier` 等，以代码为准）。
- **单花单数值**：单一印刷花色 + 单一印刷点数；**无** `double_suit`、**无** `cross_value`。
- **双花 / 跨数值**：带 **双花** 或 **跨数值**（rank 组 **8–10** 或 **JQK** 二选一）。
- **紫色钻石牌**：紫品质、单花单数值的 **紫钻档（§3.1 为 20%）**；**仅** 展示 **+💎3**（`isDiamondCard` + `diamondBonus: 3`，**无** `high_score` / `multiplier` 效果）。  
  **发钻**：该牌 **参与本手计分**（进入当手计分牌集合）时，**本手结算完成后** 立即 **`runDiamonds` +3**，并计入 **`diamondsEarnedTotal`**；**可叠加**（多手、多张各算）。**本手内即可用于商店消费**（与现 store 一致：结算后 `runDiamonds` 已更新）。

## 2. 商店：张数与品质

| 项 | 规则 |
|----|------|
| **张数** | 统一关后商店超级牌槽 **不固定**（常见 **0 / 1 / 2**），见 `Roguelike2.0统一关后商店设计.md`。无三卡品质模板。 |
| **品质（每张独立）** | **绿 30% / 蓝 40% / 紫 30%**。 |

## 3. 各品质效果与数值

### 3.0 绿、蓝

| 品质 | 单花单数值 | 双花或跨数值 |
|------|------------|--------------|
| **绿** | **+$10** 或 **+3 倍**（各 **50%**） | — |
| **蓝** | —（商店蓝牌**不产出**单花单数值） | **跨数值 50%**、**双花 50%**；各档内 **+$10** / **+2 倍**（各 **50%**）；蓝双花为 **本花 + 另三色随机**。 |

### 3.1 紫（占「已为紫牌」条件下 **100%**）

| 占比 | 内容 |
|------|------|
| **20%** | **紫钻牌**：单花，**+💎3**（仅钻字段，见 §1） |
| **40%** | **双花**（对立色）；档内 **+$20** / **+3 倍** 各 **50%** |
| **40%** | **跨数值**；档内 **+$20** / **+3 倍** 各 **50%** |

- **绿 / 蓝**（非紫钻）生成：`isDiamondCard: false`。  
- **紫钻档**：`isDiamondCard: true`，`diamondBonus: 3`。

## 6. 已定口径（答复归档）

紫单花 **+$ / +倍** 数值沿用 **+40 / +5**；双花 / 跨数值复合沿用 **+20 / +3**；参与计分定义、本手即发 💎、与 `diamondsEarnedTotal` 同步、可叠加、多钻牌各算、牌面仅 +💎3 不与 $/倍 同卡、购入价不变、**仅紫 20% 钻档** 允许 `isDiamondCard`；**局 / 关 / 手** 定义见 §0；失败局 💎 不保留局外。

---

## 7. 补充结论

- **禁牌型关不发钻**：若牌型被禁导致本手按 0 金币结算，则该手即使含紫钻计分牌也 **不发** 💎。  
- **展示不区分来源**：当前 💎 展示不区分开局赠送与本手/本关获得。
