# 超级扑克牌赎回系统设计方案

## 一、系统概述

### 1.1 核心概念
- **目标**：赎回散落在世界各地的52张超级扑克牌
- **形式**：类似关卡系统，依次解锁购买
- **效果**：超级扑克牌购买后取代白色对应牌，拥有**数值+30**的超强属性
- **激励**：每次赎回后体力回满，提供持续游戏动力

### 1.2 系统定位
- **长期目标**：为玩家提供明确的收集目标和成长路径
- **核心玩法**：通过积累积分代币，逐步解锁更强大的卡牌
- **进度可视化**：清晰展示当前进度和下一目标

### 2.2 超级扑克牌属性

#### 基础属性
- **品质**：超级品质（新增品质类型，区别于 white/green/blue/purple）
- **数值加成**：基础数值 +30
- **视觉效果**：
  - 使用 `pokers/` 文件夹中的对应图片
  - 金色闪光特效（持续动画）
  - **注意**：特效不要外发光或外描边，只在牌面内部

#### 替换机制
- **替换对象**：**仅替换白色品质的对应牌**
- **替换条件**：
  - 花色（suit）相同
  - 数值（rank）相同
  - **品质（quality）必须是 'white'**
- **替换范围**：
  - 牌池中所有符合条件的白色牌都会被替换
  - 如果有多张相同的白色牌（如3张♠️2），**全部替换**为超级♠️2
- **保留机制**：
  - **其他品质的牌保持不变**（green/blue/purple 品质的对应牌不受影响）
  - 例如：如果牌池中有绿色♠️2、蓝色♠️2、紫色♠️2，这些牌保持不变
  - 只有白色品质的♠️2会被替换为超级♠️2
- **替换时机**：购买后立即生效

### 2.3 购买机制

#### 购买条件
- **前置条件**：必须按顺序解锁，不能跳跃
- **金币要求**：拥有足够的积分代币
- **购买按钮**：金币足够时显示呼吸光效果引导点击

#### 购买效果
1. **立即效果**：
   - 扣除对应金币
   - 解锁下一张超级扑克牌
   - **体力回满**（重要激励点）
   - 替换牌池中对应的白色牌

2. **弹板显示**：
   - 标题："赎回第 n 张超级牌"
   - 牌面：显示解锁的超级扑克牌图片
   - 属性："数值+30"
   - 动画：金色闪光特效

3. **持久效果**：
   - 牌池中对应的白色牌被替换为超级扑克牌
   - 翻牌时可能抽到超级扑克牌（数值+30）

#### 价格生成函数
```typescript
function generateSuperCardPrices(): number[] {
  const prices: number[] = [];
  let currentPrice = 300;
  let increment = 200;
  
  for (let i = 0; i < 52; i++) {
    prices.push(currentPrice);
    currentPrice += increment;
    increment += 100; // 每次递增金额增加100
  }
  
  return prices;
}
```

## 四、实现要点

### 4.1 牌池替换逻辑

#### 替换时机
- 购买超级扑克牌后立即执行

#### 替换逻辑
```typescript
function replaceWhiteCardWithSuper(suit: Suit, rank: Rank) {
  // 1. 遍历牌池，找到所有符合条件的白色牌
  //    条件：suit相同 && rank相同 && quality === 'white'
  //    替换为超级扑克牌（保留原ID，只修改属性）
  //    - quality: 'white' → 'super'
  //    - baseValue: originalValue → originalValue + 30
  
  // 2. 如果玩家手中有白色对应牌，也立即替换
  //    保持牌池和手牌的一致性
  
  // 3. 注意：只替换白色品质，其他品质（green/blue/purple）保持不变
  
  // 4. 最终目标：52张全部赎回后，牌池中只有52张超级牌，没有白牌
}
```

#### 替换示例
**场景1：多张相同白色牌**
- 牌池中有：3张白色♠️2、1张绿色♠️2、1张蓝色♠️2
- 购买超级♠️2后：
  - ✅ 3张白色♠️2 → 全部替换为超级♠️2（保留原ID，修改属性）
  - ✅ 绿色♠️2 → 保持不变
  - ✅ 蓝色♠️2 → 保持不变

**场景2：不同品质的牌**
- 牌池中有：白色♠️2、绿色♠️2、蓝色♠️2、紫色♠️2
- 购买超级♠️2后：
  - ✅ 白色♠️2 → 替换为超级♠️2（保留原ID，修改属性）
  - ✅ 绿色♠️2 → 保持不变
  - ✅ 蓝色♠️2 → 保持不变
  - ✅ 紫色♠️2 → 保持不变

**场景3：已抽到手牌**
- 玩家手中：1张白色♠️2
- 牌池中：2张白色♠️2
- 购买超级♠️2后：
  - ✅ 手中的白色♠️2 → 立即替换为超级♠️2（使用pokers/poker1.png显示）
  - ✅ 牌池中的2张白色♠️2 → 全部替换为超级♠️2

**场景4：最终状态**
- 当52张超级扑克牌全部赎回后：
  - ✅ 牌池中只有52张超级牌（每张花色和数值各一张）
  - ✅ 所有白色品质的牌都被替换为对应的超级扑克牌
  - ✅ 其他品质（green/blue/purple）的牌保持不变
