# Poker Idle Game

A poker-themed idle game built with React, TypeScript, and Tailwind CSS.

## Features

- **Flip & Score**: Draw 5 cards to form poker hands and earn money.
- **Card Quality System**: Collect Green and Blue cards with special effects.
- **Auto Flip**: Unlock automation to idle.
- **Energy System**: Flips regenerate over time.

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Zustand (State Management)
- Lucide React (Icons)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

## Game Rules

- **Hands**: Standard Poker hands (High Card to Royal Flush).
- **Special Cards**:
  - Green/Blue Quality
  - High Score (+3 Value)
  - Multiplier (+2 Multiplier)
  - Double Suit (Counts as two suits)
  - Cross Value (Flexible rank)
