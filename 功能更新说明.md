# 扑克挂机游戏 - 功能更新说明

## 🎉 本次更新内容

### 1. ✅ 数据持久化系统

#### 实现内容
- **自动保存**：游戏数据自动保存到浏览器 localStorage
- **自动加载**：刷新页面后自动恢复游戏进度
- **持久化数据**：
  - 💰 积分代币（money）
  - ⚡ 剩余翻牌次数（flipsRemaining）
  - 🎴 牌池（deck）- 包括所有抽到的绿色/蓝色品质卡牌
  - 🏆 最佳手牌记录（bestHands）
  - 📊 统计数据（stats）
  - ⏰ 体力恢复时间（lastRecoveryTime）

#### 技术实现
- 使用 Zustand 的 `persist` 中间件
- localStorage key: `poker-game-storage`
- 只持久化核心数据，临时状态（如当前手牌）不保存

#### 使用说明
- **无需手动操作**：游戏会自动保存和加载
- **清除数据**：打开浏览器开发者工具 → Application/存储 → Local Storage → 删除 `poker-game-storage`
- **兼容性**：支持所有现代浏览器（Chrome、Firefox、Safari、Edge）

### 2. ✨ 动画细节优化

#### 全新动画效果

##### 🎭 牌型分级动画系统
根据牌型稀有度，提供不同级别的视觉反馈：

**传奇级（Legendary）- 皇家同花顺 👑**
- 全屏金色脉冲背景
- 旋转金色光环
- 24 个粒子爆炸特效
- 金色文字发光
- 震撼的视觉冲击

**史诗级（Epic）- 同花顺、四条 🌟💥**
- 彩虹渐变背景
- 十字光柱交叉特效
- 背景遮罩模糊
- 青色/红色主题光效

**稀有级（Rare）- 同花、顺子、葫芦 💧➡️🏠**
- 同花：花瓣飘落动画（♠️♥️♣️♦️）
- 顺子：流动光线效果
- 紫色主题边框
- 脉冲呼吸效果

**罕见级（Uncommon）- 三条、两对 ⚡✨**
- 翠绿/靛蓝主题
- 旋转缩放效果
- 适度的视觉反馈

**普通级（Common）- 一对 ⭐**
- 绿色主题
- 基础脉冲动画
- 简洁清晰

##### 💰 积分飞入动画
- 金币图标从屏幕中央飞入
- 带有旋转和缩放效果
- 金色发光拖尾
- 持续 0.8 秒的流畅动画

##### 🎨 自定义 CSS 动画
新增以下动画关键帧：
- `particle-explode` - 粒子爆炸
- `flow-light` - 流动光效
- `connect-cards` - 卡牌连接线
- `rotate-scale` - 旋转缩放
- `petal-fall` - 花瓣飘落
- `number-roll` - 数字滚动
- `coin-fly-in` - 金币飞入

#### 动画性能优化
- 使用 CSS transform 实现（GPU 加速）
- 使用 `will-change` 属性提示浏览器优化
- 动画时长优化：800ms（之前 1500ms）
- 更快的响应速度，不影响游戏节奏

## 🎯 使用建议

### 首次使用
1. 打开游戏，开始翻牌
2. 抽取几张绿色/蓝色品质卡牌
3. 刷新页面，验证数据已保存

### 体验动画
1. **快速体验**：多次翻牌，触发不同牌型
2. **传奇体验**：尝试获得皇家同花顺（概率较低，需要耐心）
3. **对比观察**：注意不同牌型的动画差异

### 性能提示
- 动画使用 GPU 加速，对性能影响极小
- 如在低端设备上卡顿，可以考虑添加"简化动画"选项（未来版本）

## 🔧 技术细节

### 数据持久化实现

```typescript
// 使用 Zustand persist 中间件
export const useGameStore = create<GameState>()(
  persist(
    (set, get) => ({
      // ... 状态和 actions
    }),
    {
      name: 'poker-game-storage',
      partialize: (state) => ({
        money: state.money,
        flipsRemaining: state.flipsRemaining,
        lastRecoveryTime: state.lastRecoveryTime,
        deck: state.deck,
        bestHands: state.bestHands,
        stats: state.stats,
      }),
    }
  )
);
```

### 动画系统架构

```
HandAnimation (主组件)
├── 背景层
│   ├── 传奇级：金色脉冲 + 旋转光环
│   ├── 史诗级：彩虹渐变 + 十字光柱
│   └── 稀有级：花瓣飘落 / 流动光线
├── 内容层
│   ├── 图标 + 名称
│   └── 积分卡片（分级配色）
└── 特效层
    ├── 粒子爆炸（24个粒子）
    └── 光柱效果
```
