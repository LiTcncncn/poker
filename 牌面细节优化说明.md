# 牌面细节优化说明

## 一、优化内容

### 1.1 添加上边距
- **问题**：卡牌内容顶到最上面，视觉上太紧凑
- **解决**：在绿蓝紫品质卡牌容器添加 `pt-2` 上边距
- **效果**：内容与边框之间有适当留白，视觉更舒适

### 1.2 双效果字体适配
- **问题**：当同时有 `+n` 和 `×n` 时，大字体会挤压或溢出
- **解决**：智能检测是否有双效果，自动调整字体大小
  - **单效果**：`text-3xl sm:text-4xl` （大字）
  - **双效果**：`text-2xl sm:text-3xl` （适中字体）
- **效果**：双效果并排显示时，字体大小适配卡牌宽度，不拥挤

### 1.3 紫色品质动态特效升级
- **问题**：原 `animate-pulse` 效果会变暗，不够绚丽
- **解决**：设计并实现新的彩虹特效
  - **边框动画**：`animate-rainbow-border` 彩虹边框循环
  - **流光特效**：`animate-shimmer-enhanced` 增强版流光

---

## 二、新增动画详解

### 2.1 彩虹边框动画（`rainbow-border`）

**效果描述**：边框颜色在紫色 → 粉色 → 蓝色 → 紫色之间平滑循环，同时外发光颜色同步变化。

**关键帧**：
```css
@keyframes rainbow-border {
  0%   { 紫色 (168, 85, 247) }
  25%  { 粉色 (236, 72, 153) }
  50%  { 蓝色 (59, 130, 246) }
  75%  { 紫色 (168, 85, 247) }
  100% { 紫色 (168, 85, 247) }
}
```

**动画参数**：
- **持续时间**：3秒
- **缓动函数**：`ease-in-out`
- **循环**：无限循环

**视觉效果**：
- 边框颜色流动变化
- 外发光（`box-shadow`）同步变色
- 内发光（`inset box-shadow`）同步变色
- 三层光效营造立体感

### 2.2 增强流光动画（`shimmer-enhanced`）

**效果描述**：彩色渐变光带从左到右扫过卡牌表面，光带颜色为紫色-粉色渐变。

**背景渐变**：
```
transparent → 紫色(0.6) → 粉色(0.8) → 紫色(0.6) → transparent
```

**动画参数**：
- **持续时间**：2秒
- **缓动函数**：`linear`
- **循环**：无限循环
- **不透明度**：动态变化（0.5 ↔ 1）

**视觉效果**：
- 光带平滑扫过
- 扫过时不透明度增强
- 彩虹色调更加绚丽

## 三、代码实现细节

### 3.1 品质样式更新

**原样式**（使用 `animate-pulse`）：
```typescript
purple: 'border-purple-500 ... animate-pulse'
```

**新样式**（使用 `animate-rainbow-border`）：
```typescript
purple: 'border-purple-500 ... animate-rainbow-border'
```

**优势**：
- 不会变暗，保持高亮度
- 彩虹色彩更加华丽
- 更强的视觉冲击力

### 3.2 流光特效层更新

**原流光**（普通 shimmer）：
```jsx
<div className="absolute inset-0 animate-shimmer opacity-30" />
```

**新流光**（增强 shimmer）：
```jsx
<div className="absolute inset-0 animate-shimmer-enhanced opacity-40" />
```

**改进**：
- 渐变色彩更丰富（紫色-粉色）
- 不透明度动态变化
- 提升到 40% 基础不透明度

### 3.3 双效果智能适配

**实现逻辑**：
```jsx
{isSpecialQuality && (() => {
  // 判断是否同时有高分和倍数效果
  const hasHighScore = card.effects.some(e => e.type === 'high_score');
  const hasMultiplier = card.effects.some(e => e.type === 'multiplier');
  const hasBothEffects = hasHighScore && hasMultiplier;
  
  return (
    <div>
      {/* 根据 hasBothEffects 动态调整字体大小 */}
      <div className={clsx(
        hasBothEffects ? "text-2xl sm:text-3xl" : "text-3xl sm:text-4xl",
        "font-black"
      )}>
        +{value}
      </div>
    </div>
  );
})()}
```

**字体大小对照**：
| 效果类型 | 字体大小（小屏） | 字体大小（大屏） |
|---------|----------------|----------------|
| 单效果   | `text-3xl`      | `text-4xl`      |
| 双效果   | `text-2xl`      | `text-3xl`      |

**间距调整**：
- 双效果时 `gap` 从 `gap-3` 减小到 `gap-2`
- 水平内边距从 `px-2` 减小到 `px-1`

### 3.4 上边距添加

**修改位置**：
```jsx
<div className="flex flex-col h-full relative z-10 pt-2">
```

**效果**：
- 花色区域不会紧贴卡牌顶部
- 整体内容向下偏移 `0.5rem`
- 视觉更加平衡

## 五、技术细节

### 5.1 CSS 关键帧动画

**彩虹边框**：
- 使用 `border-color` 和 `box-shadow` 属性
- 三层阴影：外发光（2层）+ 内发光（1层）
- 颜色同步变化，营造整体感

**增强流光**：
- 使用 `background-position` 移动渐变
- 使用 `opacity` 控制亮度变化
- `background-size: 200%` 实现平滑滚动

### 5.2 React 组件逻辑

**立即执行函数表达式（IIFE）**：
```jsx
{isSpecialQuality && (() => {
  // 逻辑代码
  return <JSX />;
})()}
```

**优势**：
- 可以在 JSX 中执行复杂逻辑
- 避免污染组件作用域
- 保持代码整洁

### 5.3 响应式设计

**字体断点**：
- 小屏（<640px）：`text-2xl` / `text-3xl`
- 大屏（≥640px）：`text-3xl` / `text-4xl`

**间距断点**：
- 统一使用 `gap-2` 和 `px-1`
- 移动端和桌面端保持一致

## 六、性能考虑

### 6.1 动画性能
- **硬件加速**：使用 `transform` 和 `opacity` 属性
- **GPU 渲染**：`box-shadow` 在现代浏览器中性能良好
- **合理帧率**：动画时长 2-3 秒，平滑但不卡顿

### 6.2 条件渲染
- 只在紫色品质卡牌上应用复杂动画
- 绿色和蓝色保持简洁高效
- 避免不必要的计算
