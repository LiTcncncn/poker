## 📊 实战收益分析

### 场景 1：一对 K + 蓝色高分牌

**卡组：** K♠️(蓝色+8) - K♥️ - 7♣️ - 5♦️ - 3♠️

**计算：**
- 参与计分：两张 K
- 数值：(10+8) + 10 = 28
- 倍数：×2（基础）
- 积分：28 × 2 = **56 分**

**对比：**
- 白色：20 × 2 = 40 分
- 提升：+40% 🎯

### 场景 2：四条 A + 蓝色倍数牌

**卡组：** A♠️ - A♥️ - A♣️ - A♦️(蓝色×4) - 3♠️

**计算：**
- 参与计分：四张 A
- 数值：10 + 10 + 10 + 10 = 40
- 倍数：×20（基础）+ 4（蓝色倍数）= ×24
- 积分：40 × 24 = **960 分**

**对比：**
- 白色：40 × 20 = 800 分
- 提升：+20% 🎯

### 场景 3：同花 + 蓝色高分 + 蓝色倍数

**卡组：** A♠️(蓝色+8) - K♠️ - Q♠️ - J♠️ - 10♠️(蓝色×4)

**计算：**
- 参与计分：全部五张

- 倍数：×10（基础）+ 4（蓝色倍数）= ×14
- 积分：58 × 14 = **812 分**

**对比：**
- 白色：50 × 10 = 500 分
- 提升：+62% 🔥

### 优先抽取

## 🔧 技术实现

### 代码变更

**1. 类型支持**
```typescript
interface CardEffect {
  type: GreenCardType;
  value?: number; // 动态数值：3, 8 (高分) / 2, 4 (倍数)
  suits?: Suit[];
  ranks?: Rank[];
}
```

**2. 抽卡逻辑**
```typescript
if (quality === 'blue') {
  const roll = Math.random();
  if (roll < 0.3) {
    // 蓝色高分 +8
    effects.push({ type: 'high_score', value: 8 });
  } else if (roll < 0.6) {
    // 蓝色倍数 +4
    effects.push({ type: 'multiplier', value: 4 });
  } else {
    // 双效果组合（绿色效果）
    // ...
  }
}
```

**3. 积分计算**
```typescript
// 动态读取 value 字段
const highScoreEffect = card.effects.find(e => e.type === 'high_score');
if (highScoreEffect?.value) {
  rankValue += highScoreEffect.value; // +3 或 +8
}
```

**4. UI 显示**
```typescript
// 根据 value 动态显示
const scoreValue = effect.value || 3;
<div className={scoreValue >= 8 ? "glow-effect" : "normal"}>
  +{scoreValue}
</div>
```
