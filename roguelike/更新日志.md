# Roguelike 更新日志

## v0.1.1（2026-05-19）

> 自上次推送（`9cb39af` 修复技能边解锁逻辑）以来的变更。

### IAA 商业化体验

- **无尽模式 IAA 计数修复**：每关 IAA 状态统一以 `run.currentStageIndex` 为键（`iaaStageKey`），不再误用 `stage.stageIndex`，避免无尽挑战中补牌/手数+1/商店 IAA 等失效或错位。
- **用尽即隐藏**：6 类 IAA 触点（补钻、商店刷新、IAA 购商品、补牌、手数+1、复活）在达到次数或前置条件不满足时**直接不显示**，不再灰显、占位或 Toast 提示。
- **商店刷新**：仅剩 💎 或仅剩 ▶ 刷新时，点「刷新」直接执行对应方式；双选项时才弹出选择层；IAA 刷新用过后不再出现在弹层中。
- **IAA 购商品**：本关已购、技能槽满、或该格已购时，该格回退为普通 💎 购买按钮。
- 新增 `utils/iaaAvailability.ts` 集中判断各 IAA 是否应展示。

### 界面与存档

- **移除装饰边框系统**：删除主界面画布整屏 `drop-shadow` 彩边、RunEntry 底部边框选择器，以及 Profile 中的 `activeFrame` / `setActiveFrame` / `getUnlockedFrames`。
- **技能附加边保留**：银/金/彩/黑边仍按主线进度在商店技能候选中解锁（`FRAME_DEFS` 仅作边池门控，不再驱动 UI 边框）。
- 新增 `GameToast` 组件与 store 字段（预留局内轻提示；IAA 限制类提示已按上条策略移除）。

### 构建

- 生产构建产物更新至 `dist/rg1/`。
